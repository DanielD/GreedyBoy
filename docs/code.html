GraphViewer ¶

Description ¶

The GraphViewer is a GUI which has the ability is to display in a simple way, different data from GreedyBoy.

These informations will be displayed:

  • Plot 1

    • Price over time (candlesticks)

    • Bot’s actions (points)

    • Bollinger bands (lines)

  • Plot 2

    • Bollinger bands width (lines and points)

    • Volatility (lines)
